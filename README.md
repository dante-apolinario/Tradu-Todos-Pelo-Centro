# Todos Pelo Centro 

Um site minimalista em Modo Escuro contendo apenas um grande botão vermelho centralizado. Ao ser pressionado, o botão reproduz a frase **"Todos Pelo Centro"** traduzida e falada em mais de 100 idiomas diferentes de forma completamente aleatória.

Este projeto utiliza a **Web Speech API (Speech Synthesis)** nativa do próprio navegador, garantindo que o áudio seja gerado em tempo real com sotaques únicos para cada dialeto, sem a necessidade de chaves de API externas ou dependências pesadas.

---

## Funcionalidades

* **Modo Escuro Nativo:** Visual limpo e confortável para os olhos (`#121212`).
* **Botão Único:** Interface extremamente minimalista e responsiva, focada em um botão de ação com efeitos visuais táteis.
* **Mais de 100 Idiomas:** Uma lista gigante de traduções inspirada nas línguas suportadas pelo ecossistema de tradução do Google.
* **Poliglota Dinâmico:** O motor de síntese tenta adaptar o sotaque para o idioma correspondente (como Japonês, Árabe, Latim, Hindi, Russo, entre outros).
* **Indicador de Idioma:** Uma barra discreta exibe o texto traduzido e o nome da língua correspondente a cada clique.

---

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica da página.
* **CSS3:** Estilização moderna, transições suaves de escala/sombra, e centralização absoluta sem dependências de frameworks.
* **JavaScript (Vanilla):** Lógica de aleatoriedade e integração direta com a API de acessibilidade do navegador (`window.speechSynthesis`).

---

## Como Executar o Projeto

Não é necessário instalar nada, configurar servidores ou rodar comandos no terminal. 

1. Faça o download ou copie o código do arquivo `index.html`.
2. Salve em uma pasta no seu computador.
3. Dê um duplo clique no arquivo `index.html` para abri-lo em qualquer navegador web moderno (Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, etc.).
4. Clique no botão vermelho e divirta-se!

## Exemplos de Idiomas Incluídos

A lista conta com uma enorme diversidade cultural, incluindo:
* **Principais Línguas Globais:** Inglês, Espanhol, Francês, Mandarim, Hindi, Russo, Árabe.
* **Línguas Regionais e Minoritárias:** Basco, Galego, Catalão, Gaélico Escocês, Sardo.
* **Línguas Africanas e Asiáticas:** Iorubá, Suaíli, Zulu, Cingalês, Tailandês, Cambojano.
* **Línguas Clássicas e Construídas:** Latim e Esperanto.
